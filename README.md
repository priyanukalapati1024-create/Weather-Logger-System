# Weather-Logger-System
**Task 4 - Slashmark LMS | INTERMEDIATE**

## 📌 Description
IoT-based weather data logging system that records temperature and humidity using DHT11 sensor and stores data in SD card for analysis. Built for Task 4 of Slashmark LMS.

## 🔧 Components Required
- Arduino Uno R3
- DHT11 Temperature & Humidity Sensor
- SD Card Module + SD Card
- Jumper Wires
- Breadboard

## ⚙️ Circuit Connections
| Component | Arduino Pin |
| --- | --- |
| DHT11 VCC | 5V |
| DHT11 GND | GND |
| DHT11 Data | D2 |
| SD Module CS | D10 |
| SD Module MOSI | D11 |
| SD Module MISO | D12 |
| SD Module SCK | D13 |
